Install
-------

Using pip
^^^^^^^^^

.. code-block:: bash

   pip install text-log-check

From release page
^^^^^^^^^^^^^^^^^

Download source code from `GitHub Releases page <https://github.com/LibSecSource/text-log-check/releases>`_

Clone from GitHub
^^^^^^^^^^^^^^^^^

.. code-block:: bash

   git clone https://github.com/LibSecSource/text-log-check.git
   make install
